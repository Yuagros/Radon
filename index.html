What Radon actually does

Target the exploit patterns that break Roblox games: movement cheats, remote spam, and invalid state.

Movement checks

Server-side validation for position, speed, and teleports so impossible movement never reaches your game loop.

  • Teleport and speed abuse blocked at the server
  • No client trust for movement

Remote guard

Rate limits and payload validation for RemoteEvents so exploit bursts and bad args are rejected before they run.

  • Throttles remote spam
  • Rejects known exploit payloads

State + evidence

Compares incoming actions to server state and writes structured logs so you have something to show when you kick or ban.

  • Invalid action chains stopped
  • Moderation-ready evidence, not raw logs
